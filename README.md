# AIVtube

AIVtube는 LLM(대형 언어 모델)과 TTS(음성 합성)를 활용한 인공지능 버추얼 유튜버(AI VTuber) 프로젝트입니다.

[📺 AIVtube 유튜브 채널](https://www.youtube.com/watch?v=fkMUewyrBaw&t=1s)  

## 🌟 Features

- **실시간 AI 버튜버 방송**
- **자연스러운 음성 생성 및 대화 가능**
- **라이브 스트리밍 지원**
- **음성 기반 인터랙션**

## 📌 결과물 예시

| Before | After |
|--------|-------|
| 💬 음성 입력 | 🎤 AI 버튜버 음성 출력 |
